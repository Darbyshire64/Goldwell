# <a href="https://blueprint.hackclub.com/projects/6683"> Goldwell </a>
Goldwell is a small form factor Solar Powered LoRA Repeater. Designed for Meshcore.

## Scope

To Design a Small Form Factor, Pole Mountable, Solar Powered Lora repeater. In Order to ensure the Node can Survive a long time without sunlight it will utilise a Samsung 18650 Battery That is undecided but with a minimum required capacity of 2000mAh to comfortabley survive the winter id idealy like a 3000mAh Battery. The Enclosure sourced from RAKwirless has a built in Solar Pannel.
